# Short.Vue

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Run Tests:

```
yarn test
```

<b> Unit </b> testing:

```
yarn test:unit
```

<b> End-To-End </b> testing:

```
yarn test:e2e
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
